# 1-Wire Raspberry Pi Adapter

A KiCad project for a simple interface board connecting 1-Wire sensors to a Raspberry Pi.

This hardware is part of the [boilert](https://github.com/guycorbaz/boilert) project, which monitors boiler temperatures using a Raspberry Pi.

## Features

- **Raspberry Pi Integration**: Designed for the standard 40-pin Raspberry Pi GPIO header.
- **1-Wire Connectivity**: Includes Molex Micro-Fit 3.0 connectors for 1-Wire sensors and external power.
- **Integrated Bus Support**: Schematic includes the necessary pull-up resistor and power filtering.
